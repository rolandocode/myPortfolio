Open ERP Software

Open ERP its an open source project tath implements Angular 8 and Net 5. Deployed in Microsoft Azure App Service. Also, data is storage in MS SQL with Azure Database

This repository includes:

  • Net 5 application: Net 5 application with MVC and service components architecture, project can easily migrate to Net 6, endpoints can be only reached throught JWT security token. This project also includes swagger page
  • Angular 8 client: Angular 8 client with all user interface and on guard security

Deployed solution: https://open-erp-admin.azurewebsites.net/
